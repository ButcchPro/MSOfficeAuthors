# MS Office Authors

Простое приложение WPF для управления авторами документов MS Office.

## Как запустить

1. Убедитесь, что у вас установлен .NET SDK.
2. Склонируйте репозиторий.
3. Откройте папку в Visual Studio Code.
4. Выполните команду `dotnet run` в терминале или используйте задачу VS Code "Run Application".

## Структура проекта

- `MainWindow.xaml`: Главное окно приложения.
- `MSOfficeAuthors.csproj`: Файл проекта .NET.
